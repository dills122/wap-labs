Normative target

Strict core. Explicit extensions.

The first compatibility target is the exact WAP-215 Class C client graph applied to the WAP 1.2.1 effective specification, with WML 1.3 observable behavior as the markup floor.

class-c-data-clientSelected From Wap 215

primary strict target

Exact CCR-CLASSC-C-001 data-client profile: WAE, WML, WBXML, WMLScript, WMLScript libraries, caching, WSP, WDP, and WCMP mandatory client features. WTP becomes mandatory only when connection-mode WSP is supported; security, push, telephony, identity, and UAProf remain separately declared.

class-b-telephony-clientDeferred

optional future profile

Adds mandatory telephony, push, and security profile behavior after strict Class C closure.

class-a-complete-clientDeferred

optional future profile

Makes mandatory and optional release features mandatory; not part of the first compliance release.

waves-enhancedPlanned

additive extension mode

Modern features and improved behavior that preserve strict results or are explicitly separated from strict mode.

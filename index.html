WAP Emulator Stack For Deterministic Engine Work

A pre-alpha layered emulator stack focused on WML deck/card fidelity, strict layer boundaries, and reproducible behavior across parser, runtime, and host integration.

Open Simulator

Stack Layers

  • Gateway: kannel environment + protocol behavior
  • Transport: WBXML and constrained transport APIs
  • Engine: Rust/WASM WML parser, runtime, layout, navigation
  • Desktop Host: simulator harness and integration adapters

Current Focus

  • Deterministic deck/card navigation semantics
  • Stable wasm contract for host integration
  • Repeatable fixture-driven engine behavior
  • Manual and scripted validation through simulator timelines

The simulator includes curated fixtures, metadata-driven acceptance checklists, runtime state snapshots, and per-example event tracking with export.
